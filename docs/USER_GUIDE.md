# دليل المستخدم — CipherOwl

## مقدمة

CipherOwl هو مدير كلمات مرور ذكي من الدرجة العسكرية، يوفر حماية بيومترية مستمرة وتشفير بدون معرفة مسبقة (Zero-Knowledge).

## التثبيت

### Android
1. حمّل التطبيق من Google Play Store
2. افتح التطبيق واتبع خطوات الإعداد الأولي

### iOS
1. حمّل التطبيق من App Store
2. افتح التطبيق وأنشئ حسابك

### Windows / macOS / Linux
1. حمّل المثبت من الموقع الرسمي
2. ثبّت التطبيق واتبع التعليمات

## الإعداد الأولي

### إنشاء الحساب
1. **كلمة المرور الرئيسية**: اختر كلمة مرور قوية (12+ حرف مع رموز وأرقام)
2. **مفتاح الاسترداد**: احفظ عبارة BIP-39 المكونة من 24 كلمة في مكان آمن
3. **المصادقة البيومترية**: سجّل بصمة الوجه للدخول السريع

### إعداد المصادقة الثنائية (2FA)
1. اذهب إلى **الإعدادات** ← **الأمان** ← **المصادقة الثنائية**
2. امسح رمز QR بتطبيق المصادقة
3. أدخل رمز التحقق المكون من 6 أرقام

## إدارة كلمات المرور

### إضافة كلمة مرور
1. اضغط على **+** في الشاشة الرئيسية
2. أدخل اسم الموقع / التطبيق
3. أدخل اسم المستخدم وكلمة المرور
4. اضغط **حفظ**

### مولد كلمات المرور
- **الطول**: 8-128 حرف (يُنصح بـ 20+)
- **خيارات**: أحرف كبيرة/صغيرة، أرقام، رموز خاصة
- **عبارات المرور**: كلمات عشوائية مفصولة بشرطة

### البحث والتصفية
- استخدم شريط البحث للعثور على كلمات المرور بسرعة
- صنّف بالفئات: شبكات اجتماعية، بنوك، بريد، إلخ

## الحماية البيومترية

### Face-Track (تتبع الوجه المستمر)
- يراقب وجهك باستمرار أثناء استخدام التطبيق
- يقفل تلقائياً إذا ابتعدت عن الشاشة
- يكشف محاولات الانتحال بتحليل الملمس والرمش

### فتح القفل
- **بصمة الوجه**: الطريقة الأسرع
- **بصمة الإصبع**: بديل سريع
- **كلمة المرور الرئيسية**: دائماً متاحة
- **Windows Hello**: على أجهزة Windows المتوافقة

## المزامنة السحابية

- تشفير كامل قبل الرفع (Zero-Knowledge)
- مزامنة تلقائية عبر الأجهزة
- مؤشر حالة المزامنة في الشاشة الرئيسية
- يعمل بدون اتصال مع طابور العمليات المعلقة

## مركز الأمان

### نقاط القوة
- تقييم قوة كل كلمة مرور
- كشف كلمات المرور المكررة
- تنبيهات كلمات المرور القديمة

### أكاديمية الأمان
- دروس تفاعلية في أمن المعلومات
- نظام نقاط ومكافآت (gamification)
- شارات إنجاز لتحفيز الممارسات الآمنة

## الخصوصية وحذف الحساب

### حذف الحساب (GDPR)
1. اذهب إلى **الإعدادات** ← **الحساب** ← **حذف الحساب**
2. اقرأ التنبيه بعناية
3. أكّد بكتابة "حذف"
4. يتم حذف جميع البيانات نهائياً من الخادم

# User Guide — CipherOwl

## Introduction

CipherOwl is a military-grade smart password manager with continuous biometric protection and zero-knowledge encryption.

## Installation

### Android
1. Download from Google Play Store
2. Open the app and follow the initial setup

### iOS
1. Download from the App Store
2. Open the app and create your account

### Windows / macOS / Linux
1. Download the installer from the official website
2. Install and follow the on-screen instructions

## Initial Setup

### Account Creation
1. **Master Password**: Choose a strong password (12+ characters with symbols and numbers)
2. **Recovery Key**: Save your 24-word BIP-39 phrase in a safe location
3. **Biometric Authentication**: Enroll your face for quick access

### Two-Factor Authentication (2FA)
1. Go to **Settings** → **Security** → **Two-Factor Authentication**
2. Scan the QR code with your authenticator app
3. Enter the 6-digit verification code

## Managing Passwords

### Adding a Password
1. Tap **+** on the home screen
2. Enter the website/app name
3. Enter username and password
4. Tap **Save**

### Password Generator
- **Length**: 8–128 characters (20+ recommended)
- **Options**: Uppercase/lowercase, digits, special symbols
- **Passphrases**: Random words separated by dashes

### Search & Filter
- Use the search bar for quick lookup
- Categorize: social media, banking, email, etc.

## Biometric Protection

### Face-Track (Continuous Monitoring)
- Continuously monitors your face while the app is in use
- Auto-locks if you move away from the screen
- Detects spoofing via texture analysis and blink detection

### Unlock Methods
- **Face Recognition**: Fastest method
- **Fingerprint**: Quick alternative
- **Master Password**: Always available
- **Windows Hello**: On compatible Windows devices

## Cloud Sync

- Full encryption before upload (Zero-Knowledge)
- Automatic sync across devices
- Sync status indicator on the home screen
- Works offline with pending operations queue

## Security Center

### Strength Scoring
- Evaluates the strength of each password
- Detects duplicate passwords
- Alerts for old/stale passwords

### Security Academy
- Interactive cybersecurity lessons
- Points and rewards system (gamification)
- Achievement badges to encourage secure practices

## Privacy & Account Deletion

### Account Deletion (GDPR)
1. Go to **Settings** → **Account** → **Delete Account**
2. Read the warning carefully
3. Confirm by typing "حذف" (delete)
4. All data is permanently removed from the server
